### Hi there 👋

I'm **Theodor Solbjorg** — a software engineer with more than twenty years behind the keyboard. I started my software journey in **2006** and have been working with **AI since 2013**, long enough to have watched the field go from a quiet specialty to the thing everyone is suddenly building on.

Over the years I've worked across a wide stack — C# .NET, Delphi, Python, Assembly, Visual Basic, C/C++, JavaScript, HTML5, CSS3, ASP, PHP, SQL and more — with a long-standing love of languages, low-level systems, and plug-'n'-play architectural designs.

### 🤖 What I'm working on now

My focus these days is **AI and large language models** — specifically the parts that decide whether a model is actually useful in production:

- **Generative workflows** — chaining models into pipelines that do real, multi-step work.
- **Dense information systems** — getting the right context to the model so it reasons over what matters.
- **AI tooling** — retrieval, evaluation, agents, and the scaffolding that turns a capable model into something dependable.
- **Model architectures** — how these systems are built underneath, and where the next gains are hiding.

### 🛠️ A bit of background

A few things I've built along the way:

- **An operating system** with a hybrid kernel — built mostly to understand the abstractions I relied on.
